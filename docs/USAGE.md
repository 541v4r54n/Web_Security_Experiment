# 使用说明（USAGE）

## 1. 平台功能

- 用户管理：注册、登录/登出、个人信息维护、账号删除
- 管理员功能：用户列表、删除用户（首次注册用户默认成为 Admin）
- 会话管理：基于 Cookie Session（支持超时）
- 用户操作记录：登录、登出、资料修改、上传等都会记录
- 核心业务：多用户图片水印管理（上传图片 → 生成水印版 → 在线查看/下载）
- Web攻防实验：SQL 注入、命令注入（“漏洞版 / 防御版”对比）

## 2. 推荐体验流程

1. 注册一个用户并登录
2. 上传一张图片并填写水印文字
3. 到“操作记录”查看日志
4. 到“攻防实验”页面分别体验：
   - SQL 注入：漏洞版检索 vs 防御版检索
   - 命令注入：漏洞版 ping vs 防御版 ping

## 3. HTTPS 说明

本项目使用自签名证书，仅用于实验。浏览器提示“不受信任”属于正常现象。
